Génération d'un installer pour une version X.Y.Z
1: Avoir compilé le projet micro_brico_poc1 depuis Qtcreator en mode release 
2: editer dans le gen_installer.bat la version et la date tout en haut. 
3: lancer une console windows et executer le setqtenv.bat dans le dossier depuis la console
4: lancer le script gen_installer.bat depuis la console
